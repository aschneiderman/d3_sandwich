How to Select Stuff in D3 Using CSS "Selectors"

NOTE: at some point need to turn this into a polished page that the tutorial. For now, it's a place to dump my notes as I learn more about how to use the hooks that are built into CSS for selecting elements on a webpage with the help of what are called CSS "selectors."

Examples

d3.selection('form input[type="text"]')
d3.selection('form input[type="submit"]'), Which gets you the submit button
d3.selection('Label[for="contact"]')'
d3.selection('input[type=checkbox]')
d3.selection(".button-all, .button-large")
