Step #2: Add the State Names to the Map

Now that we've created one rectangle for each state, we need to label the states. And just like creating the rectangles, it only takes a few lines of code.

What we want to do is to take each object in the state_map array and use its dataa to create a text label. Here's the code that gets it done.

Before we break it down, take a quick look at the whole thing. It should look a little familiar: it's basically a variation of the same technique we used to create the rectangles.

And yeah, the first few lines -- "selectAll", "data", "enter", "append" -- stil look weird. :)

Remember our motto: don't sweat it. You'll see this technique used over and over in D3; once you've played with a couple recipes, it'll be almost second nature. 


var  state_map = [
    { "state": "ME", "row": 0, "col": 10 },
        { "state": "VT", "row": 1, "col": 9 },
        { "state": "NH", "row": 1, "col": 10 },
        ...

    ];

    // Put the name of each state on its rectangle
    svg_area.selectAll("text")
      .data(state_map)
      .enter()
      .append("text")
        .attr("x", function(d,i)  { return d.col * width + 108;})
        .attr("y", function(d,i) {return d.row * height +  26;})
        .text(function (d) { return d.state; })
        .style("font",   "16px sans-serif")
        .style("fill", "White")
        .style("pointer-events", "none");

OK, let's break it down:

A) Use the Weird Trick to Create One Text Label per State

.selectAll: Select all of the text labels already on the page (Even though there are none)
.data: Get ready to connect each text label with the data for that state: 'state', 'row', and 'col'.
.enter: For every state which doesn't already exist (i.e., all of them)...
.append: add a new text label
