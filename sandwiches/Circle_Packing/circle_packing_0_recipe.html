Circle Packing

Although you can create just about anything you want using D3, D3 also has some built-in tools to make it easier to do some pretty data visualizations. Circle Packing is one of the simplest to use that produces pretty impressive results.

NOTE: this recipe is based on Mike Bostock's Circle Packing block

Difficulty: Beginner

Prerequsites: Abstract State Map

The Recipe

1) Create the CSS styles for the circles

2) Create the JSON to store the info about the data for each node

3) Generate the data about where to items on the screen and create a group to contain the circle and text for each point of data

4) Add the circle and text to each group
